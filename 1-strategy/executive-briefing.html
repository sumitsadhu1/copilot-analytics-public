2. Decisions You Need to Make

2.1 Pre-Configuration Discovery Questions

Before your IT team touches any configuration, you should be able to answer these:

#QuestionWhy It Matters
1How many qualifying Viva Insights service plans are assigned (not merely purchased), and how many are Microsoft 365 Copilot?Microsoft’s Environment Requirements article explicitly counts 30 Copilot plus 20 Viva Insights as 50 Viva service plans, while the Dashboard article says 50 Copilot or 50 Viva Insights. Verify assigned service plans and tenant availability; use 50 Copilot plus agent activity as the conservative Agent Dashboard prerequisite.
2Is the Entra ID manager hierarchy populated for most users?Determines automatic vs. manual dashboard access. If >20% of users lack a Manager attribute, auto-identification and Manager Settings will be unreliable.
3Do you have an HR data export available (department, function, location)?Required for group-level filtering and BU-level rollups. See Data Strategy.
4Has anyone been assigned the AI Administrator role?Required to manage dashboard access and privacy settings
5Do you have compliance/legal requirements to exclude specific users?Drives exclusion list configuration (5-day processing)
6Has Copilot already been rolled out, or is it pre-deployment?Before/after analysis needs 4 weeks of pre-adoption baseline
7Do you use Viva Glint or Viva Pulse for employee surveys?Enables sentiment page with filters & heat maps
8What is the primary reporting need: Copilot-only or broader productivity?Determines whether standalone Viva Insights license is needed
9Are you deploying Copilot agents / Copilot Studio agents?Determines whether Agent Dashboard section is relevant
10Who are the target report consumers (IT admins, BU leaders, analysts)?Drives which tiers to prioritize

2.2 Decision Framework

  1. Do you need user-level license utilization data? → Start with Admin Center Reports
  2. Do you have 1+ assigned Copilot licences AND want ROI metrics? → Enable Copilot Dashboard (50+ for advanced features)
  3. Do leaders want ready-to-use targeted reports? → Copilot Analytics Reports (auto-available with Dashboard)
  4. Are you deploying agents (Copilot Studio, etc.)? → Validate and enable the Agent Dashboard (preview)
  5. Do you need to monitor Copilot Credits by service or organisational scope? → Use the Consumption Dashboard, then reconcile with official billing records
  6. Do you need before/after adoption analysis or custom segmentation? → Configure Advanced Analytics
TIP: Progressive Implementation
Most customers benefit from implementing this guide’s framework progressively. Start with admin center reports, add the Copilot Dashboard and Analytics Reports, validate the Agent and Consumption dashboards where relevant, then add Advanced Reporting for custom analysis.

5. Key Concepts for Leaders

5.1 Copilot Assisted Hours & Value

The primary ROI metric. Composed of three categories:

CategoryWhat's countedAssistance factor
Meeting SummariesDuration of each meeting summarized by CopilotActual meeting duration (variable)
Search & SummariesChat prompts, email/doc summaries, analysis6 minutes per action
Content CreationDraft email, Draft in Word, Create presentation6 minutes per action

Assisted Value = Assisted Hours × Hourly Rate (default $72/hour, configurable per session).

Diagnostic coverage and causality

Optional diagnostic data affects Copilot assisted hours, emails sent using Copilot, Teams meetings and meeting hours summarized or recapped, and Teams chats summarized. Required diagnostic data still supplies a subset, so disabled optional diagnostics can undercount these measures. Confirm the tenant’s diagnostic-data policy and metric coverage before comparing groups or periods.

Assisted Hours and Assisted Value are modeled estimates based on Microsoft’s assumptions and a session-only hourly rate. They do not prove realized savings or that Copilot caused a business outcome. Use a documented baseline, stable diagnostic coverage, and business outcome data before making ROI or causal claims.

Source: Connect to the Microsoft Copilot Dashboard for Microsoft 365 customers (Last updated 1 July 2026), Impact section.

5.2 Licensing — What You Get with Copilot

Every M365 Copilot licence automatically includes the Viva Insights service plan at no additional cost. That bundled plan contributes to Viva Insights eligibility, but each capability still has its own assigned-plan threshold, role, access, data, cloud, and preview requirements. A separate Viva Insights purchase is needed when the measured population includes employees who do not receive the bundled plan through Copilot.

5.3 Privacy by Design

  • No individual tracking: All dashboard data is aggregated at group level
  • Minimum group size: Default 10 (configurable down to 5)
  • Differential privacy: Automatically applied to protect individuals
  • Exclusion lists: Remove specific users from all analytics

5.4 Known Metric Underreporting

Microsoft has acknowledged two data collection issues. No action needed — data auto-corrects after the affected windows.

Affected MetricsAffected Period
Email features (drafts, summaries)June 1, 2025 – February 18, 2026
Meeting features (summaries, recap)September 6 – November 3, 2025

6. Scope & Limitations

In Scope

  • Microsoft 365 Copilot applications: Word, Excel, PowerPoint, Outlook, Teams, Loop, OneNote, Edge, Copilot Chat
  • Six official capability areas: admin center reports, Copilot Dashboard, Agent Dashboard, Consumption Dashboard, Copilot Analytics reports, and Advanced Reporting
  • License utilization, ROI measurement, sentiment analysis, agent adoption

Out of Scope

  • Other Copilot products: Copilot for Sales, Copilot for Service, GitHub Copilot, Security Copilot
  • Copilot Chat users without a Microsoft 365 Copilot licence: covered by the current Microsoft 365 admin center Microsoft 365 Copilot Chat usage report and, when enabled and available, the Copilot Dashboard’s Copilot Chat view. Microsoft Graph’s licensed Copilot usage endpoint remains licensed-user only; use the admin report or appropriate Purview audit source for unlicensed scenarios.
  • Broader Viva Insights productivity metrics (meeting time, focus time) — see Collaboration Analysis Guide
Current Copilot Chat reporting

The Microsoft 365 admin center provides a dedicated usage report for Copilot Chat users without a Microsoft 365 Copilot licence, including exportable user activity. The Copilot Dashboard also provides a Copilot Chat licence view when reporting is enabled and available. These are different from licensed-user Microsoft Graph usage reporting and from Purview audit evidence.

Source: Microsoft 365 Copilot Chat usage report (Last updated 10 July 2026) and Microsoft 365 Copilot reporting options for admins (Last updated 16 September 2025).
