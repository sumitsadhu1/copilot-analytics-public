Microsoft 365 Copilot Analytics — Quick-Start Cheat Sheet

Implementation Checklist • Six Copilot Analytics Areas • Validated 16 July 2026

Admin Center Reports (Day 1)

  • Verify Reports Reader role
  • Reports → Usage → Microsoft 365 Copilot
  • Review licensed Copilot and unlicensed Copilot Chat reports separately
  • Wait for report processing → export CSV if needed

Copilot Dashboard (Day 1–7)

  • Core processing: 1+ Copilot or 50 assigned Viva Insights service plans
  • 50-plan capabilities: count the Viva service plan included with Copilot; verify conflicting Learn wording in the tenant
  • Verify Entra ID manager hierarchy populated
  • AI Admin: add self + users to access
  • Insights Admin: enable Manager Settings (toggle on "Grant managers scoped access")
  • Use Organizational Data in Microsoft 365 unless Message Center and tenant UI show another supported path
  • Configure minimum group size (default 10, min 5)
  • Enable optional diagnostics for comprehensive Assisted Hours inputs
  • Document the $72/h estimate and reapply custom rate/currency each session
  • Optional: exclusion list, sentiment surveys
  • Wait up to 7 days → verify Readiness, Adoption, Impact, scope, and filters

Copilot Analytics Reports

  • No action — auto-available to Dashboard users
  • Verify: Viva Insights → Reports (left nav)

Agent Dashboard (Preview)

  • Conservative prerequisite: 50 Copilot licences + agent activity
  • Not available in GCC/national-regional clouds
  • Delegates need "Your company" scope (not "Your group")
  • Validate feature presence and preview data handling in the tenant

Advanced Reporting (Week 2+)

  • Assign Insights Analyst role → wait 1–2 hours
  • Grant partition access
  • Insights Admin and Insights Analyst roles are independent
  • Install a current Power BI Desktop version
  • Create Copilot Impact query
  • Connect Power BI via OData feed → apply template
  • Use Business Impact for outcome correlation; do not claim causation

Consumption Dashboard

  • Global-partition analyst or Global Admin: company scope
  • Enabled manager: group scope; no delegation
  • Current M365 service data: Cowork and Work IQ API
  • Reconcile official charges in Microsoft 365 admin center/GitHub

Common Gotcha: AI Admin does NOT get automatic dashboard access — must add self via access management settings.
Multi-agency: Partitions scope analyst queries. They are not tenant or cryptographic isolation. Test Dashboard, Agent, exports, delegates, managers, global roles, Power BI, and global partitions before go-live.
