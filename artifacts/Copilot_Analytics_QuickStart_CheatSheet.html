Microsoft 365 Copilot Analytics — Quick-Start Cheat Sheet

Implementation Checklist • All 5 Tiers • February 2026

Tier 1 — Admin Center (Day 1)

  • Verify Reports Reader role
  • Reports → Usage → Microsoft 365 Copilot
  • Wait ~72 hours for data → export CSV

Tier 2 — Copilot Dashboard (Day 1–7)

  • Verify 50+ assigned Copilot/Viva Insights licences
  • Verify Entra ID manager hierarchy populated
  • AI Admin: add self + users to access
  • Insights Admin: enable Manager Settings (toggle on "Grant managers scoped access")
  • Upload org data CSV (PersonId, ManagerId, Organization + custom)
  • Configure minimum group size (default 10, min 5)
  • Configure hourly rate for ROI (default $72/h)
  • Optional: exclusion list, sentiment surveys
  • Wait up to 7 days → verify all 4 pages

Tier 3 — Analytics Reports (Auto)

  • No action — auto-available to Dashboard users
  • Verify: Viva Insights → Reports (left nav)

Tier 4 — Agent Dashboard (Day 7+)

  • On by default — verify in Viva Insights left nav
  • Delegates need "Your company" scope (not "Your group")
  • If agents deployed, verify activity data

Tier 5 — Advanced Analytics (Week 2+)

  • Assign Insights Analyst role → wait 1–2 hours
  • Grant partition access
  • Org data uploaded & processed (from Tier 2)
  • Install Power BI Desktop (June 2022+)
  • Create Copilot Impact query
  • Connect Power BI via OData feed → apply template
  • Configure scheduled refresh
