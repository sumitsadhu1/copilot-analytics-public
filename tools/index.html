Start with the question, not the tool

Use this page to distinguish consumption monitoring, adoption exports, customizable queries, Microsoft Graph reports, and audit sources. Each has a different population, permission model, data shape, and authority.

3. Quick Decision Checklist

  1. Define whether the question is about billing, consumption, adoption, impact, value, or audit.
  2. Define the required population: licensed Copilot users, unlicensed Copilot Chat users, a partition, a manager group, or all covered events.
  3. Confirm the least-privileged role or API permission and the supported cloud environment.
  4. Record the source, scope, period, refresh timestamp, identifiers, and privacy treatment with every output.
  5. Reconcile against the authoritative surface: billing portal for charges, source report for adoption, or Purview for audit events.
  6. Do not combine sources until metric definitions, time boundaries, and identity handling are explicitly mapped.
